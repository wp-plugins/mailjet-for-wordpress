=== Mailjet for Wordpress ===
Contributors: mailjet
Tags: emailing
Requires at least: 3
Tested up to: 3.3.1
Stable tag: 1.0.1

Reconfigures the wp_mail() function to use Mailjet SMTP instead of mail() and creates an options page to manage the settings.

== Description ==

Mailjet is the best way to send and track emails in real time, while ensuring their deliverability.

Take advantage of our reporting tools: you will get advanced statistics to monitor and optimize your emails.
To further fulfill your needs, our terrific API and dedicated module for WordPress is at your disposal.

== Installation ==

1. Log in as administrator in Wordpress.
2. Go to Extensions > Add and send `mailjet-for-Wordpress.zip`.
3. Active the Mailjet extension and go to the settings page.
